## 一、前言

本部分内容是关于Java并发的一些知识总结，既是学习的难点，同时也是面试中几乎必问的知识点。

面试中可能会问的一些问题：

- 创建线程的方式
- Synchronized/ReentrantLock 
- 生产者/消费者模式
- volatile关键字
- 乐观锁/悲观锁
- 死锁
- 了解的并发集合

因此针对以上问题，整理了相关内容。
