## 一、字符串

关于字符串的题目，在面试和笔试中都有很多出现。主要包括以下几个方面：

1. **大数问题。**
2. **字符串的翻转（全部翻转、部分翻转），拼接等问题。**
3. **字符串的模式匹配，找重复子串、公共前缀、回文问题。**

以上几个方面的题目在剑指offer以及leetcode动态规划方面都有出现过。本部分主要整理了几个比较典型常考的题目。
